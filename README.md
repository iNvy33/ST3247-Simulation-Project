# ST3247-Simulation-Project

The final data set I used is made up of 3 generation sessions, each saved to a .npz file

To recreate the data, run `simulating.py` 3 times

Run 1:
Change value in create_tables(___) to 5000
Change file name to simulation_results.npz

Run 2:
Change value in create_tables(___) to 5000
Change file name to simulation_results_1.npz

Run 3:
Change value in create_tables(___) to 10000
Change file name to simulation_results_2.npz
